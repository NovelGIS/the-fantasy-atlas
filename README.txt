Fantasy Character Tracker Map

An interactive map visualizing character locations across fantasy series by chapter.

Built using Leaflet and GeoJSON, this project allows readers to follow character movements throughout the story in a spatial, chapter-by-chapter format.

✨ Features

    Character locations displayed on an interactive map
    Chapter-based filtering
    Book selection dropdown
    Next / Previous chapter navigation
    Toggleable character labels
    Marker clustering for improved readability

🛠️ Built With

    HTML, CSS, JavaScript
    Leaflet.js
    GeoJSON
